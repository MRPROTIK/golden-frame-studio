## Tech Stack

- Pure HTML / CSS / JS — zero frameworks, zero dependencies
- Google Fonts: Playfair Display, Cormorant Garamond, Montserrat
- CSS backdrop-filter glassmorphism
- Animated aurora background (pure CSS keyframes)
- IntersectionObserver scroll fade-in animations
- Fully mobile responsive

## Deploy to GitHub Pages
```bash
git clone https://github.com/MRPROTIK/golden-frame-studio
cd golden-frame-studio
cp ~/Downloads/golden_frame_studio_aurora.html index.html
git add index.html
git commit -m "Add luxury demo website"
git push origin main
```

Then: **Settings → Pages → Deploy from main → / (root)**
